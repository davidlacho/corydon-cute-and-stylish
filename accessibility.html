Accessibility Statement

♿ Inclusive Design • 📅 Updated March 26, 2026

At Stay Winnipeg, we aim to make this site usable for as many people as possible. We design and test pages with accessibility in mind so guests can browse accommodation details, local guides, and booking links with confidence.

Our Accessibility Goal

We target conformance with the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA across the site. This includes clear structure, readable contrast, keyboard access, and meaningful text alternatives for media.

What We Build In

  • Skip links and clear page landmarks so keyboard and assistive technology users can jump to main content quickly.
  • Semantic HTML heading hierarchy for predictable screen reader navigation.
  • Descriptive image alternative text for content images and empty alt text for decorative visuals.
  • Keyboard-accessible controls with visible focus styles for menus, links, and interactive elements.
  • Colour and contrast checks to keep text readable on all core layouts.

Tools We Rely On

We combine automated and manual checks as part of development and maintenance:

  • Pa11y (WCAG 2.1 AA): Automated accessibility audits for key pages and full-sitemap sweeps.
  • WAVE and browser accessibility tooling: Additional checks for structure, contrast, and ARIA issues.
  • Keyboard-only testing: Tab, Shift+Tab, and Escape behaviour checks for navigation and overlays.
  • Manual content reviews: Headings, link labels, alt text quality, and readable page flow.

Safeguards Before Deployment

Before publishing new versions of the site, we run a release checklist that includes accessibility gates:

  • Run WCAG 2.1 AA checks for page-level changes and shared layout/component updates.
  • Resolve blockers such as contrast failures, duplicate IDs, missing iframe titles, and missing ARIA labels.
  • Re-test affected pages after fixes to verify regressions are removed.
  • Keep supporting SEO and discovery metadata up to date so users and assistive technology can find accurate content.

Continuous improvement: Accessibility is ongoing work. We review findings from audits and user feedback, then prioritize fixes in upcoming site updates.

Need Help Accessing Content?

If you run into an accessibility barrier on this website, please contact us through our Airbnb listing and include the page URL and a short description of the issue. We will do our best to address it quickly and provide an accessible alternative.
